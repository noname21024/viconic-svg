## 📄 Collection Metadata

Each `info.json` contains:

```json
{
  "prefix": "ti",
  "info": {
    "name": "Tabler Icons",
    "url": "https://tabler-icons.io",
    "total": 4000,
    "author": {
      "name": "Paweł Kuna",
      "url": "https://github.com/tabler/tabler-icons"
    },
    "license": {
      "title": "MIT License",
      "spdx": "MIT",
      "url": "https://opensource.org/licenses/MIT"
    }
  }
}
```
